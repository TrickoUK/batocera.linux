# INCLUDED-EMUS.md

Emulators expected to be built into the `x86_64-arcade` custom target
(`configs/batocera-x86_64-arcade.board`), as currently configured
(arcade systems only — Nvidia excluded, PS1/PS2 not yet enabled).

This was generated by reading the already-resolved
`output/x86_64-arcade/.config` (produced by an earlier `make
x86_64-arcade-config` run) and cross-referencing it against the package
definitions in `package/batocera/emulators/` — no build/config commands
were run to produce this list, so it doesn't touch or interfere with any
build currently in progress. If the board file changes, this list should
be regenerated the same way.

## Standalone emulators

These run as their own programs, not through RetroArch:

- **MAME** (`BR2_PACKAGE_MAME`) — the standalone/upstream MAME arcade emulator.
- **Supermodel** (`BR2_PACKAGE_SUPERMODEL`) — Sega Model 3 arcade emulator.
- **Flycast** (`BR2_PACKAGE_FLYCAST`) — multi-platform Sega Dreamcast, Naomi
  and Atomiswave emulator. Pulled in for Naomi/Atomiswave arcade board
  support (arcade selects Sega Naomi/Atomiswave support, which brings in
  the Dreamcast-family emulators).
- **Redream** (`BR2_PACKAGE_REDREAM`) — alternative Dreamcast/Naomi/Atomiswave
  emulator, included alongside Flycast.
- **Hypseus Singe** (`BR2_PACKAGE_HYPSEUS_SINGE`) — laserdisc arcade game
  emulator (Dragon's Lair, Space Ace, and similar).

## RetroArch + libretro cores

**RetroArch** (`BR2_PACKAGE_RETROARCH`) is the frontend/core-loader itself.
The following are the libretro "cores" that plug into it:

- **MAME** (`BR2_PACKAGE_LIBRETRO_MAME`) — current MAME as a libretro core.
- **MAME2003-Plus** (`BR2_PACKAGE_LIBRETRO_MAME2003_PLUS`) — older/leaner
  MAME core, better suited to lower-spec hardware.
- **FinalBurn Neo** (`BR2_PACKAGE_LIBRETRO_FBNEO`) — arcade emulator core
  (CPS1/CPS2/CPS3, Neo Geo, and many other arcade boards).
- **Flycast** (`BR2_PACKAGE_LIBRETRO_FLYCAST`) — libretro-core version of
  Flycast (Dreamcast/Naomi/Atomiswave), alongside the standalone build above.
- **DICE** (`BR2_PACKAGE_LIBRETRO_DICE`) — emulates arcade games built from
  discrete logic circuits (no CPU at all).
- **Same CDi** (`BR2_PACKAGE_LIBRETRO_SAME_CDI`) — Philips CD-i emulator
  core (MAME-derived).

## Not emulators, but included in support of the above

- `BR2_PACKAGE_BATOCERA_BEZEL`, `BR2_PACKAGE_BATOCERA_SHADERS`,
  `BR2_PACKAGE_COMMON_SHADERS`, `BR2_PACKAGE_GLSL_SHADERS`,
  `BR2_PACKAGE_SLANG_SHADERS` — bezel artwork and shader packs for RetroArch.
- `BR2_PACKAGE_LIBRETRO_CORE_INFO`, `BR2_PACKAGE_RETROARCH_ASSETS` —
  RetroArch's core-info database and UI assets/themes.
- `BR2_PACKAGE_SUPERMODEL_COMMON` — shared data/library package for Supermodel.

## Notably absent (by design, at this phase)

- Any Nvidia driver package (excluded via
  `board/batocera/x86/local-patches/no-nvidia.patch`).
- PS1/PS2 (`BR2_PACKAGE_DUCKSTATION`, `BR2_PACKAGE_PCSX2`) — commented out
  in the board file, planned for phase 2. See `USER-INSTRUCTIONS.md` for
  how to add them (or any other system) back in.
- Everything else `BATOCERA_ALL_SYSTEMS` would normally pull in (N64,
  GameCube, handhelds, home computers, DOS, ScummVM, Wine/Windows games,
  homebrew, GameStream, ports, Flash) — none of those categories are
  enabled in this board file yet.
